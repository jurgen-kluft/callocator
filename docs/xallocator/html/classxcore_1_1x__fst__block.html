Detailed Description

x_fst_block contains an array of x_fst_elem objects. This is the smallest memory chunk which is allocated from system (via allocated/deallocator) and is the smallest unit by which the allocator can grow. The inElemSize and inNumElements parameter sent to init function determines the size of the block.
