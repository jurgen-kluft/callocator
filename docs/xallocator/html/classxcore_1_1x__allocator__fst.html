Detailed Description

x_allocator_fst is a fast allocator for objects of fixed size.

It preallocates (from ) blocks with T elements. By calling Allocate an application can fetch one T object. By calling Deallocate an application returns one T object to pool. When all objects on the pool are used, pool can grow to accommodate new requests. specifies by how many blocks the pool will grow. Reset reclaims all objects and reinitializes the pool. The parameter RestoreToInitialSize can be used to resize the pool to initial size in case it grew.

Note:
This allocator does not guarantee that two objects allocated sequentially are sequential in memory.
